An Jia Website - Vercel Deployment

This page provides a simple way to deploy the An Jia website to Vercel, with a focus on fixing the properties page data fetching issue.

1Login to Vercel

First, make sure you're logged in to your Vercel account:

Login to Vercel

2Go to Your Project

Navigate to your An Jia project on Vercel:

Go to Dashboard

3Create a New Deployment

Click on "Deployments" in the sidebar, then click the "Deploy" button.

4Update Environment Variables

Make sure the following environment variables are set in your Vercel project:

NEXT_PUBLIC_WORDPRESS_API_URL=http://wp.ajyxn.com/wp-json
WORDPRESS_API_URL=http://wp.ajyxn.com/wp-json
WORDPRESS_FALLBACK_API_URL=http://199.188.200.71/wp-json
WORDPRESS_DIRECT_API_URL=http://wp.ajyxn.com/wp-json

You can update these in your project settings under "Environment Variables".

5Verify the Deployment

After deployment, verify that the properties page is working correctly:

  1. Visit https://ajyxn.com/properties
  2. Check that property data is loading correctly
  3. Open the browser console (F12) and look for any errors
  4. Test the property filters and pagination

6Troubleshooting

If issues persist after deployment:

  • Check the Vercel deployment logs for any errors
  • Verify that the WordPress API is accessible at http://wp.ajyxn.com/wp-json
  • Check for CORS errors in the browser console
  • Ensure that the Content Security Policy in vercel.json is correctly configured
